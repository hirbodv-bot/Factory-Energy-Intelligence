# Factory Energy Intelligence

An interactive Streamlit application for analysing manufacturing energy cost, production performance, downtime, defects, carbon emissions, anomalies, and energy-saving scenarios.

## Features

- Built-in demonstration dataset
- Excel upload and downloadable template
- Date filtering
- Interactive Plotly dashboards
- KPI cards
- Energy anomaly detection
- Automated engineering insights
- Cost and energy-saving scenario analysis
- Indicative simple payback
- Excel and CSV export

## Deployment

This repository can be deployed directly using Streamlit Community Cloud.

- Branch: `main`
- Main file: `app.py`

## Disclaimer

This is a preliminary engineering decision-support tool. Verify all data, assumptions, equations, tariffs, emissions factors, and site conditions before using the outputs for operational or investment decisions.
